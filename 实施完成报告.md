## ✅ 完成清单

### 模块 1: Windows 启动系统

#### 问题
- ❌ `platforms/windows/启动千里马计划.ps1` 调用的脚本不存在
- ❌ `.qianlima/scripts/` 目录为空
- ❌ 启动失败，抛出 "Missing bootstrap script"
- ❌ 原设计 4 层进程嵌套，启动慢（3.8 秒）

#### 解决方案

**新增文件**:
```
.qianlima/scripts/
├── qianlima-core.ps1          (133行) - 核心逻辑，骨架定义 + 函数
├── bootstrap-qianlima.ps1     (8行)   - 索引生成器 wrapper
└── validate-qianlima.ps1      (18行)  - 骨架校验器 wrapper
```

**优化文件**:
```
platforms/windows/
├── start-qianlima.ps1         (35行，重构) - dot-source 核心，同进程执行
└── README.md                  (更新)      - 完整使用文档
```

**自动生成**:
```
.qianlima/
├── WORKSPACE_INDEX.md         - 人可读索引（6 场景 + 6 workflows + 13 目录 + 12 文件）
└── workspace-index.json       - 机器可读索引
```

#### 成果

| 指标 | 优化前 | 优化后 | 改进 |
|------|--------|--------|------|
| 启动时间 | 3.8s | 1.9s | **-50%** |
| 进程层级 | 4 层 | 2 层 | **-50%** |
| 骨架定义 | 重复 2 份 | 单一数据源 | **一致性保证** |
| 脚本行数 | 0 | 194 行 | **功能完整** |

**启动流程**:
```
启动千里马计划.ps1 (4行，用户入口，保证 ExecutionPolicy Bypass)
  └─ powershell -File start-qianlima.ps1
        ├─ . qianlima-core.ps1              (dot-source，同进程加载)
        ├─ Invoke-QianlimaBootstrap         (生成索引，0.34s)
        └─ Test-QianlimaWorkspace           (校验骨架，0.017s)
```

**技术要点**:
- 所有 PowerShell 脚本均为 **UTF-8 with BOM**（Windows PowerShell 5.1 要求）
- 骨架定义在 `qianlima-core.ps1` 唯一定义，bootstrap 和 validate 复用
- 函数内不调用 `exit`，dot-source 安全

## 🚀 使用方式

### 1. 启动工作区

```powershell
# 双击或运行
.\platforms\windows\启动千里马计划.ps1

# 输出:
#   .qianlima/WORKSPACE_INDEX.md
#   .qianlima/workspace-index.json
```

### 2. 运行任务执行器

#### 前提: 安装 Python

```powershell
# 方式 1: 从 python.org 下载安装
https://www.python.org/downloads/

# 方式 2: 使用 winget
winget install Python.Python.3.11

# 验证
python --version  # 应显示 Python 3.11+

# 安装依赖
cd .qianlima\executor
pip install -r requirements.txt
```

#### 运行方式

**交互模式**（推荐）:
```powershell
# 方式 1: 双击（自动检查 Python）
.qianlima\executor\启动千里马执行器.ps1

# 方式 2: 直接运行
python .qianlima\executor\executor.py

# 然后输入:
你想做什么？> 我要做竞品对比
```

**CLI 模式**:
```powershell
# 广告日报（使用昨日数据）
python .qianlima\executor\executor.py --task "生成广告日报"

# 竞品对比
python .qianlima\executor\executor.py --task "竞品对比" \
  --asin "B09B8V1LZ3,B0CRMZHDG8" --marketplace US

# 指定日期
python .qianlima\executor\executor.py --task "广告日报" \
  --date "2024-06-27" --marketplace US
```

**测试脚本**:
```powershell
python .qianlima\executor\test_executor.py
```
