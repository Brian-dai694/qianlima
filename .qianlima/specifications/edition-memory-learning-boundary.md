# Personal and Enterprise Memory Boundary

## Personal Qianlima

Personal memory reduces interaction friction. Explicit preferences are user-editable and deletable. Observed habits and local experiences remain shadow candidates until repeated behavior and user confirmation promote them. They may improve interaction and read-only routing only; they never alter permissions or safety confirmations.

## Enterprise North Star

Enterprise memory stores approved policy, verified knowledge, receipt-bound task evidence, and sanitized improvement candidates. Access is constrained by tenant, department, role, project, and task. Learning requires replay, independent verification, human approval, monitored release, and rollback.

## Boundary

Personal memory never becomes enterprise knowledge. Enterprise knowledge never becomes a private profile. Enterprise policy may constrain a personal deployment, but it does not write into personal habit memory.
