### 季节性成因推断

- **11-12月**：黑五 + 圣诞礼物季 + 年末项目收尾采购 + 冬季室内焊接项目增多
- **1-3月**：冬季室内项目持续 + 美国退税季（购买力释放）+ 新年设备更新
- **4-5月**：春季维修/农耕/户外施工启动
- **6-8月**：夏季户外活动优先，焊接需求季节性下降
- **9-10月**：秋季复工，为冬季项目备货

## 二、新品上架时间表

### 黄金窗口：9月1日-10月15日 入仓可售

```
7月 → 确认供应商，下样品单
8月 → 样品验证 + 大货下单 + 海运发货（25-35天）
9月 → 入仓 FBA + Listing 上线 + Vine 索评启动
10月 → 评论积累 + 广告冷启动 + 自然排名爬升
11月→ 旺季放量
```

| 阶段 | 时间窗口 | 动作 | 目标 |
|---|---|---|---|
| **选品决策** | 6-7月 | 调研完成，供应商确认 | 锁定产品和价格 |
| **样品验证** | 7月 | 打样 + 实测 + 专利排查 | 品质过关 |
| **大货生产** | 8月1-15日 | 下单 300-500 件首单 | 工厂排期 |
| **海运** | 8月15日-9月15日 | 美森/以星快船 18-25天 | 到港 |
| **入仓** | 9月15-30日 | FBA 入仓 + 可售 | 赶上旺季前奏 |
| **冷启动** | 10月 | Vine 索评 + 低预算广告 + Coupon | 积累前 15-20 条评论 |
| **旺季放量** | 11月-12月 | 广告预算翻倍 + 促销 | 抢占旺季流量 |
| **稳定运营** | 1-3月 | 维持排名 + 补货 | 旺季延续收割 |

### 次优窗口：2月1日-3月1日 入仓可售

```
11月 → 下单生产
12月 → 海运（避开圣诞港口拥堵）
1月 → 到港入仓
2月 → 冷启动
3月 → 旺季收尾 + 退税季
```

- 适用于：错过了 9 月窗口的产品
- 风险：旺季已过大半，但仍有退税季购买力
- 优势：12月海运价低，入仓快

### 不推荐窗口

| 窗口 | 原因 |
|---|---|
| **4-5月入仓** | 入仓后即进入淡季，评论积累期与需求低谷重叠，冷启动成本高 |
| **6-8月入仓** | 全年最淡，广告转化率低，Vine 索评慢 |

### 安全库存公式

```
旺季安全库存（11-3月）= 日均销量 × 旺季系数 1.3 × (生产周期30天 + 海运25天 + 入仓7天 + 缓冲15天)
淡季安全库存（6-8月）= 日均销量 × 淡季系数 0.8 × (生产周期30天 + 海运25天 + 入仓7天 + 缓冲15天)
```

### 补货节奏表

| 时间 | 动作 | 补货量 | 目的 |
|---|---|---|---|
| **8月15日前** | 🟢 下单旺季补货 | 覆盖 11-3 月销量的 70%（海运到仓） | 旺季库存到位 |
| **10月1日** | 🟡 检查旺季库存 | 如 11-12 月预售超预期，紧急补空运 | 黑五/圣诞不缺货 |
| **12月15日** | 🟢 下单次年春季补货 | 覆盖 1-5 月销量 | 利用 12 月海运低价 |
| **3月1日** | 🟡 淡季前库存检查 | 降库存至淡季水平，避免长期仓储费 | 控制库存成本 |
| **5月15日** | 🟢 下单淡季补货（小批量） | 覆盖 6-9 月 | 维持可售不断货 |
| **7月1日** | 🔴 旺季备货启动 | 大货下单，覆盖 10-3 月 | 全年最重要的补货节点 |
