# 大众任务卡

任务卡是给普通用户使用的入口。用户不需要知道数据源、工具、workflow 或脚本，只要选择自己要做的事。

使用方式：

```text
我要做竞品对比。
我要优化一个 Listing。
我要测算一个产品利润。
我要看关键词排名。
我要找一个新品机会。
```

每张任务卡都说明：

- 适合什么时候用
- 用户需要提供什么
- Agent 会做什么
- 最终会产出什么
- 哪些动作不会自动执行
- 使用哪种跨文件、跨项目和模型交接协议

任务卡里的 `communication_protocol` 是给 Agent 看的。普通用户不需要理解这些字段；它们用于让 Agent 少读无关文件、用引用和摘要传递上下文、记录来源，并把 token 和使用情况写入台账。
