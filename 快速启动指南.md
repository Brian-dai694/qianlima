# 千里马计划 - 快速启动指南

> 5 分钟上手千里马计划任务执行器

## 🚀 第一步：启动工作区

**Windows 用户**：
```powershell
# 双击或运行
.\platforms\windows\启动千里马计划.ps1
```

**输出**：
- ✅ `.qianlima/WORKSPACE_INDEX.md` - 工作区索引
- ✅ `.qianlima/workspace-index.json` - 机器可读索引
- ✅ 校验 13 个目录、12 个治理文件

**用时**: ~1.9 秒

---

## 🐍 第二步：安装 Python（如果尚未安装）

### 检查是否已安装

```powershell
python --version
```

如果显示版本号（如 `Python 3.11.x`），跳到第三步。

### 安装 Python

**方式 A - 官方安装器**（推荐）:
1. 访问 https://www.python.org/downloads/
2. 下载 Python 3.11+ 安装包
3. 安装时**勾选** "Add Python to PATH"

**方式 B - winget**:
```powershell
winget install Python.Python.3.11
```

### 验证安装

```powershell
python --version
# 应显示: Python 3.11.x 或更高
```

---

## 📦 第三步：安装依赖

```powershell
cd ".qianlima\executor"
pip install -r requirements.txt
```

**安装的包**：
- `pyyaml` - YAML 解析
- `anthropic` - Claude API（预留）
- `jinja2` - 模板渲染（预留）
- `python-dotenv` - 环境变量管理

## ❓ 常见问题

### Q1: `python: command not found`
**A**: Python 未安装或未添加到 PATH。重新安装 Python 并勾选 "Add Python to PATH"。

### Q2: `ModuleNotFoundError: No module named 'yaml'`
**A**: 依赖未安装。运行 `pip install -r requirements.txt`。

### Q3: PowerShell 脚本中文乱码
**A**: 终端编码问题（不影响脚本执行）。可运行：
```powershell
[Console]::OutputEncoding = [System.Text.Encoding]::UTF8
```

### Q4: 任务匹配失败
**A**: 检查输入是否包含任务关键词。可用关键词：
- 竞品、竞品对比
- listing、优化
- 利润、测算
- 关键词、排名
- 新品、选品
- 广告、日报

### Q5: 数据显示"模拟"
**A**: 当前 MCP 工具和飞书 API 使用模拟数据。实际数据源需配置：
1. 飞书 API token → `secrets.local.yaml`
2. MCP 工具集成 → 需在 Claude Code 环境中运行
