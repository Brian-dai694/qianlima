# LINGMA.md — 通义灵码 / Qoder CN 使用入口

通义灵码或 Qoder CN 打开本仓库后，先按 `QODER.md` 工作。本文是面向通义灵码的短入口。

## 必读文件

```text
QODER.md
README.md
DESKTOP_AGENT_BRIEF.md
.qianlima/WORKSPACE_INDEX.md
.qianlima/templates/realtime-cost-card_template.md
```

## 适合做的事

- 公开模板工程维护
- 规则、workflow、task-card、模板整理
- PowerShell 脚本修复
- GitHub Actions / CI 检查
- 隐私剔除和 Git-safe 校验
- 成本卡、usage ledger、节约规则维护

## 不适合做的事

- 真实 ERP 操作
- 真实广告竞价或预算修改
- 真实价格、采购、Listing 写回
- 未脱敏业务数据处理

## 输出要求

非简单任务必须输出：

- 当前场景
- 已读取来源
- 修改文件
- 验证结果
- 成本状态卡
- 待确认项
