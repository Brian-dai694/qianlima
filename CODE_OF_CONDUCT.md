# 行为准则 · Code of Conduct

## 我们的承诺

为营造开放友好的环境，我们承诺让每个参与者的体验不受骚扰，无论其经验水平、性别、性取向、身心状况、外貌、体型、种族、年龄、宗教或国籍。

## 可接受的行为

- 使用友善、包容的语言
- 尊重不同观点与经验
- 接受建设性批评
- 以社区利益为重

## 不可接受的行为

- 骚扰、侮辱、人身攻击
- 公开或私下的骚扰行为
- 未经许可发布他人隐私信息
- 在 issue / PR 中提交他人的真实运营数据或凭据

## 执行

违反准则的行为可通过 GitHub 私密渠道向仓库所有者报告。维护者有权删除、编辑或拒绝不符合本准则的评论、提交、代码和其他贡献。

本准则改编自 [Contributor Covenant](https://www.contributor-covenant.org) v2.1。
