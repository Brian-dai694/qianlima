## ⚠️ 重要提示

### 编码要求
所有 PowerShell 脚本必须保持 **UTF-8 with BOM** 编码：
- qianlima-core.ps1
- bootstrap-qianlima.ps1
- validate-qianlima.ps1
- start-qianlima.ps1
- 启动千里马执行器.ps1

**原因**: Windows PowerShell 5.1 按系统 ANSI 编码（GBK）解析 .ps1 文件，UTF-8 without BOM 会导致中文字面量报错。

**重新编码方法**:
```powershell
$content = Get-Content -LiteralPath <file> -Raw -Encoding UTF8
Set-Content -LiteralPath <file> -Value $content -Encoding UTF8
```

### 数据源配置
当前使用模拟数据。要使用真实数据：
1. 创建 `.qianlima/secrets.local.yaml`（参考 `secrets.example.yaml`）
2. 配置飞书 API token
3. 配置 MCP 服务器（需在 Claude Code 环境）
